# L1 score32 exp-LUT schedule-wrapper PPA

This proposal measures bounded 2-cluster and 4-cluster RTL wrappers that combine
central command dispatch, local ready/valid issue, local completion accounting,
and the selected score32 exp-LUT 8x8 ppc1 dual-stream datapath.

The result should be used as a bounded schedule/control composition cost. It is
not a claim that the full 856-replica Llama7B array has been placed.
