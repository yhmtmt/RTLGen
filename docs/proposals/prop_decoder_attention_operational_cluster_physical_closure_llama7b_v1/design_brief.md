# Operational Cluster Physical Closure

The current precision-aligned separated two-pass candidate reaches 1595.421 us
per token in the planning model, but its dense compute cost comes from a narrow
self-stimulating hash harness. The next promotion gate replaces that transfer
with an operational accumulating tile and maps the 16 MiB score store into an
explicit macro banking network.

The physical hierarchy is intentionally staged:

1. Prove and measure one operational 16x8 signed-int8 tile.
2. Prove and place one 16Kx256 score bank made from eight depth groups and seven
   parallel `fakeram45_2048x39` slices per group.
3. Compose two heads, two score banks, approximately 106-108 operational tiles,
   and shared versus per-head iterative dividers in one cluster.
4. Promote only routed cluster survivors into a 16-cluster top-level NoC study.

`fakeram45_2048x39` has usable Nangate45 LEF and Liberty views but no GDS in the
installed platform. Every result using it is therefore a floorplan, routing,
and timing proxy, not SRAM implementation signoff.
